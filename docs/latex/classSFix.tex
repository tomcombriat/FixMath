\doxysubsection{Detailed Description}
\subsubsection*{template$<$int8\+\_\+t NI, int8\+\_\+t NF, uint64\+\_\+t RANGE$>$\newline
class SFix$<$ NI, NF, RANGE $>$}

Instanciate an signed fixed point math number. 
\begin{DoxyParams}{Parameters}
{\em NI} & The number of bits encoding the integer part. The integral part can range into \mbox{[}-\/2$^\wedge$\+NI, 2$^\wedge$\+NI -\/1\mbox{]} \\
\hline
{\em NF} & The number of bits encoding the fractional part \\
\hline
{\em RANGE} & A purely internal parameter that keeps track of the range used by the type. Safer to {\itshape not} define it. \\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Note}
The total number of the underlying int will be NI+\+NF+1 in order to accomodate the sign. It follows that, if you want something that reproduces the behavior of a int8\+\_\+t, it should be declared as SFix$<$7,0$>$. 
\end{DoxyNote}
